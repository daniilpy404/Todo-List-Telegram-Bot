# 📝 Todo-List Telegram Bot

Простой бот для управления задачами с использованием Python и SQLite.

---

# ⚙️ Функции

- Добавление задач
- Удаление задач
- Отметка выполнения
- Отдельные задачи для каждого пользователя


# 🧠 Стек-технологии

- Python
- aiogram
- SQLite


# 🚀 Запуск

```bash
python main.py
